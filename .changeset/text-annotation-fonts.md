Rework the text annotation tool. Text shapes gain a font family picker (the same set Ghost loads from fonts.bunny.net), bold/italic toggles, alignment buttons, and a font-size control in the panel. Clicking an existing text annotation with the text tool re-opens its editor instead of stacking a new one, and the inline editor no longer shows a duplicate selection frame while editing. Text size is set from the panel rather than by dragging handles.
